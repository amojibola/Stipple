What this prompt is
Use this when starting a new Claude Code session in the middle
of a layer. It combines the briefing, the status block, and
the handoff note into one paste so Claude Code has everything
it needs to pick up exactly where the last session stopped.

When to use it
Any time you are resuming work mid-layer in a new Claude Code
session.

How to use it
1. Copy the full briefing document from your notes
2. Copy the current status block from your notes
3. Copy the handoff note from your last session
4. Open Claude Code and paste everything in this order:

   FIRST:  Full briefing document
   SECOND: Status block
   THIRD:  This connector text below (fill in the blanks)
   FOURTH: The handoff note from last session

---

THE CONNECTOR TEXT TO PASTE BETWEEN STATUS BLOCK AND HANDOFF NOTE:

We ended the last session mid-layer. Here is the exact handoff
note from where we stopped. Use this to understand what exists,
what is working, and what to build next. Do not rebuild anything
already completed. Do not skip ahead.

Read the handoff note carefully and then tell me:
1. What files already exist that you will need to work with
2. What the very next thing to build is
3. Confirm this matches the handoff note before writing any code
